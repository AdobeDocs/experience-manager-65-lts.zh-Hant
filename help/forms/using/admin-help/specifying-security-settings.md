# 指定安全性設定 {#specifying-security-settings}

>[!NOTE]
> 
> 確保使用者具有存取管理員控制檯的管理員許可權。

Forms可讓您控制是否解析XML輸入中的外部實體。 預設會解決問題，但您可以變更此行為以提高AEM表單系統的安全性。

**禁止處理包含外部實體參照的XML資料檔**

1. 在管理控制檯中，按一下&#x200B;**[!UICONTROL 服務> Forms]**。
1. 清除「解析外部圖元」核取方塊。
1. 按一下「**[!UICONTROL 儲存]**」。
